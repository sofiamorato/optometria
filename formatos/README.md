# Formatos Clínicos

Esta carpeta contiene todos los documentos oficiales utilizados actualmente por la clínica.

Estos documentos son la fuente de verdad del proyecto.

La aplicación debe adaptarse a estos formatos.

Nunca modificar los formatos para adaptarlos a la aplicación.

---

## Historia_Primera_Vez_Vacia.pdf

Plantilla oficial para consultas de primera vez.

Debe utilizarse como referencia para generar el PDF final.

---

## Historia_Primera_Vez_Ejemplo.pdf

Ejemplo diligenciado.

Se utiliza para comprender cómo se llenan los campos.

---

## Historia_Control_Vacia.pdf

Plantilla oficial para consultas de control.

---

## Historia_Control_Ejemplo.pdf

Ejemplo diligenciado de una consulta de control.

---

## Certificado_Ejemplo.pdf

Ejemplo de certificado.

La estructura sirve como base para certificados, órdenes, remisiones y otros documentos similares.

---

## Reglas

Los documentos de esta carpeta nunca deben modificarse.

Si se agregan nuevos formatos, deben conservar su nombre descriptivo y mantenerse organizados.
